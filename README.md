# openclaw-kmb-bus-arrival
Retrieves real-time KMB bus arrival information using the official Hong Kong government transport API.

Contents:

kmb-bus-arrival/
├── kmb_bus.py          (main executable)
├── fixed_kmb_bus.py    (alternative/fixed version)
├── kmb_bus.sh          (shell wrapper)
├── SKILL.md            (OpenClaw skill definition)
├── README.md           (documentation)
└── test.js             (tests)


### To Install/Upload to Clawhub:

1. Download the tar.gz file from the server
2. Extract: tar -xzf kmb-bus-arrival-v1.1.0.tar.gz
3. Copy the kmb-bus-arrival folder to your OpenClaw skills/ directory
4. Restart OpenClaw or reload skills
